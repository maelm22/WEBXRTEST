Welcome to the WebXR Test Site

This site contains links to three different WebXR test environments. Each test is designed to evaluate how WebXR interacts with various renderers created in Godot. Please choose one of the following options to begin your testing experience:

Test Environments

  • Compatible Test

    This test evaluates the compatibility of WebXR with standard rendering techniques in Godot.

  • Forward Test

    This test focuses on the forward rendering approach in Godot and its performance under WebXR.

  • Mobile Test

    This test assesses how well WebXR performs on mobile devices using Godot's rendering capabilities.
